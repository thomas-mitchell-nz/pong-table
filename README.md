This repository contains the code I developed for a light-up pong table that I designed and built. The table itself consists of cup detecting RGB pods on each side of the table. Each pod contains 3 RGB LEDS and a pair of infra-red emitters and receivers. The electronics are interfaced with via an Arduino MEGA. All of the electronics are hidden from the players under a sheet of opal perspex. I hope to build on this project in the future.
